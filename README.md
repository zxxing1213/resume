# Java开发工程师简历模板

这是一个专为Java开发工程师设计的现代化简历模板，采用HTML+CSS构建，支持在线查看和打印输出为PDF格式。

## 项目特色

- **现代化设计**：采用Tailwind CSS框架，响应式设计，美观专业
- **打印优化**：特别针对打印输出进行了优化，确保PDF格式完美呈现
- **内容丰富**：包含个人信息、专业技能、工作经历、项目经验等完整模块
- **易于定制**：所有内容均可轻松修改，适配不同求职者的需求

## 简历结构

### 1. 个人信息
- 姓名、职位、联系方式
- 工作年限、年龄、学历、证书等关键信息
- 个人总结：突出核心优势和职业定位

### 2. 专业技能
- Java核心技术（JVM、并发编程等）
- Spring Cloud Alibaba微服务生态
- 数据库架构与调优
- Camunda工作流引擎
- AI工程化落地经验
- 全栈开发能力
- DevOps实战经验

### 3. 工作经历
按时间倒序列出工作经历，每段经历包括：
- 公司名称和任职时间
- 工作职责和成就
- 使用技术和解决方案

### 4. 项目经验
详细介绍代表性项目：
- 项目名称和角色
- 技术栈标签
- 项目描述
- 核心贡献和成果

## 使用方法

1. **在线查看**：
   直接在浏览器中打开 `index.html` 文件即可查看简历效果

2. **打印为PDF**：
   - 在浏览器中打开 `index.html`
   - 使用浏览器的打印功能（Ctrl+P 或 Cmd+P）
   - 选择"另存为PDF"选项
   - 调整页面设置（建议A4纸张，边距适中）
   - 保存PDF文件

3. **自定义内容**：
   - 使用文本编辑器打开 `index.html`
   - 根据注释提示修改个人信息、工作经历等内容
   - 保存后即可看到更新效果

## 技术实现

- **HTML5**：语义化标签，结构化内容
- **Tailwind CSS**：现代化样式框架，响应式设计
- **Font Awesome**：图标库，增强视觉效果
- **CSS打印样式**：专门的@media print规则，确保打印效果

## 兼容性

- 支持所有现代浏览器（Chrome、Firefox、Safari、Edge）
- 完美支持A4纸张打印
- 移动端友好，可自适应不同屏幕尺寸

## 注意事项

- 请确保在打印前预览效果，调整合适的缩放比例
- 建议使用最新版浏览器以获得最佳效果
